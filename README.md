# Simulador PoW com Tkinter

## Instalação

1. Instale as dependências do projeto:
   pip install -r requirements.txt

2. OBS: A biblioteca Tkinter já vem com o Python, mas em algumas distros Linux pode ser necessário instalar manualmente:
   sudo apt-get install python3-tk

## Execução

Para rodar o simulador, execute:
   python main.py
